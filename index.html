About This Project

This GeoNarrative a web-based storytelling application that explores popular study locations across the University of Washington campus. It demonstrates how spatial data and interactive web maps can help students discover environments that match their study needs.

This project was created as part of GEOG 458: Digital Geography using Mapbox GL JS and Scrollama.
